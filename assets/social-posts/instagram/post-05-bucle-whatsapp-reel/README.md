# Reel 05 — El Bucle del WhatsApp

## Qué son estos archivos

Los 8 archivos SVG de esta carpeta son **frames base** para montar el Reel en Canva o CapCut. No son un vídeo — son las diapositivas que se importan como imágenes estáticas y se animan.

## Duración sugerida por frame

| Frame | Duración |
|-------|----------|
| frame-01.svg | 2 segundos |
| frame-02.svg | 3 segundos |
| frame-03.svg | 3 segundos |
| frame-04.svg | 2 segundos |
| frame-05.svg | 2 segundos |
| frame-06.svg | 2 segundos |
| frame-07.svg | 3 segundos |
| frame-08.svg | 4 segundos |

**Total: ~21 segundos** (duración ideal para Reel con alta retención)

## Cómo montarlo en Canva

1. Abre Canva y crea un diseño nuevo de vídeo **1080 × 1920 px**
2. Exporta primero cada SVG a PNG (ver sección exportación en el README principal)
3. Importa los 8 PNG como imágenes
4. Añade cada imagen como diapositiva con la duración indicada arriba
5. Aplica una transición rápida (fade o ninguna) entre frames
6. Añade música de fondo: instrumental minimalista, sin letra, volumen bajo
7. Exporta como MP4

## Cómo montarlo en CapCut

1. Crea un proyecto nuevo en vertical (9:16)
2. Importa las imágenes en orden (frame-01 → frame-08)
3. Ajusta duración de cada clip según la tabla
4. Añade audio de fondo
5. Exporta en 1080p
