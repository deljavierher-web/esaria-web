Reglas de uso

✓ Hacer

  • Mantener proporciones originales del logo
  • Respetar el margen mínimo alrededor del logo
  • Usar sobre fondos con contraste suficiente
  • Usar el isotipo para favicon y avatares
  • Usar el logo horizontal para la web

✗ Evitar

  • Deformar o estirar el logo
  • Cambiar los colores sin criterio
  • Usar sobre fondos con poco contraste
  • Añadir sombras, efectos 3D o degradados
  • Recrear el logo con otra tipografía
