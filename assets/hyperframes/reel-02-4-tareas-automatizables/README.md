## Previsualizar

```bash
cd assets/hyperframes/reel-02-4-tareas-automatizables
npx hyperframes preview
```

Abre: `http://localhost:3002/#project/reel-02-4-tareas-automatizables`

## Renderizar a MP4

```bash
cd assets/hyperframes/reel-02-4-tareas-automatizables
npx hyperframes render
```

Output: `renders/reel-02-4-tareas-automatizables_<timestamp>.mp4`

Para calidad final:

```bash
npx hyperframes render --fps 30 --quality high
```

## Requisitos

- Node.js >= 22
- FFmpeg (`brew install ffmpeg`)
- No requiere `package.json` propio — se ejecuta vía `npx`
