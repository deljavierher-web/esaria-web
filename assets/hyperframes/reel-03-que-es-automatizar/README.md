# Reel 03 — ¿Qué es automatizar un negocio?

Reel vertical 9:16 para Instagram / TikTok explicando qué es automatizar un negocio de forma simple y sin tecnicismos.

## Especificaciones

- **Formato:** 1080 × 1920 px (9:16)
- **Duración:** 24 segundos
- **Motor:** HyperFrames + GSAP
- **Paleta:** #0F172A · #6D5EF3 · #FFFFFF · #94A3B8 · #A090F5

## Escenas

| # | Escena | Tiempo |
|---|--------|--------|
| 1 | Hook — "Automatizar un negocio no es poner robots" | 0–3 s |
| 2 | Problema — "Es dejar de repetir tareas manuales" + pills | 3–7 s |
| 3 | Ejemplo 01 — cita / recoger datos | 7–11 s |
| 4 | Ejemplo 02 — pregunta frecuente / asistente | 11–15 s |
| 5 | Idea clave — "No se trata de usar IA porque sí" | 15–19 s |
| 6 | CTA — "¿Qué tarea repites cada semana?" | 19–24 s |

## Comandos

```bash
# Previsualizar en el studio
cd assets/hyperframes/reel-03-que-es-automatizar
npx hyperframes preview

# Lint antes de renderizar
npx hyperframes lint

# Renderizar a MP4 (calidad estándar)
npx hyperframes render

# Renderizar en alta calidad para entrega
npx hyperframes render --fps 30 --quality high --output renders/reel-03-que-es-automatizar.mp4
```

## Salida

El MP4 queda en `renders/` con timestamp automático o en la ruta indicada con `--output`.
